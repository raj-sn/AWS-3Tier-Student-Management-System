# Lessons Learned

Through this project I learned:

- How to design a custom VPC
- How subnets work
- Difference between public and private subnets
- How route tables control network traffic
- Purpose of Internet Gateway
- How security groups work
- How Amazon EC2 hosts applications
- How Amazon RDS stores application data
- How PHP connects to MySQL databases
- How Application Load Balancers distribute traffic
- Purpose of Target Groups
- Importance of Availability Zones
- AWS troubleshooting techniques
- Basic cloud architecture design principles

## Key Takeaway

Building the project provided practical experience in networking, security, database management, troubleshooting, and cloud architecture deployment on AWS.
